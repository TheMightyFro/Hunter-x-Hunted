\# Hunter x Hunted



Hunter x Hunted is a fast-paced 1v1 local multiplayer chase game built in Python using Pygame.  

One player becomes the \*\*Hunter\*\*, the other is the \*\*Hunted\*\*. Outrun, outplay, or dash through moving obstacles as you fight to survive three intense rounds.



Designed as a competitive couch game with simple controls, dynamic power-ups, destructible effects, and a polished HUD.



---



\## Features



\- \*\*Local 2-Player Versus Mode\*\*

\- \*\*3-Round Match System\*\* with scoreboard and round summaries

\- \*\*Dynamic Moving Obstacles\*\* with collision physics

\- \*\*Moving Power-Ups\*\*  

&nbsp; - Speed Boost  

&nbsp; - Slow Debuff  

\- \*\*Dash Mechanic\*\* (obstacle-phasing, cooldown system, visual aura)

\- \*\*Sprint + Stamina Bars\*\*

\- \*\*Hunger Phase\*\* (mid-round difficulty spike)

\- \*\*Particle Effects, Slow-Mo Hits, Screen Shake, Flash FX\*\*

\- \*\*Menus \& UI\*\*

&nbsp; - Title Screen

&nbsp; - HUD Buttons (Menu, Restart, Controls, Music, SFX, Fullscreen)

&nbsp; - Round End \& Match End Panels

\- \*\*Separate Menu + Game Music\*\*



---



\## Controls



\### Player 1

\- \*\*Movement:\*\* W A S D  

\- \*\*Sprint:\*\* Left Shift  

\- \*\*Dash:\*\* Left Alt  



\### Player 2

\- \*\*Movement:\*\* Arrow Keys  

\- \*\*Sprint:\*\* Space  

\- \*\*Dash:\*\* Right Alt  



---



\## Keyboard Shortcuts

\- \*\*ESC\*\*: Toggle Fullscreen  

\- \*\*M/S/F buttons on HUD\*\* for Music, SFX, Fullscreen  

\- \*\*Mouse:\*\* Click HUD buttons  

\- \*\*ENTER:\*\* Start game / next round
