Hunter x Hunted

A fast-paced local-multiplayer chase game built in Python using Pygame.

Hunter x Hunted is a competitive split-role arena game where one player becomes the Hunter and the other becomes the Hunted. Outmaneuver, outpace, and outsmart your opponent as roles reverse, timers tick down, and the arena fills with obstacles, effects, and power-ups. Designed for quick rounds, high replayability, and couch multiplayer fun.

Features

Local Multiplayer (2 Players)
Fast hunter-vs-hunted gameplay with distinct movement and sprinting systems.

Dynamic Roles & Round Flow
Smooth intro transitions, round timer, and halftime speed changes.

Full UI / HUD System
Countdown timer, role labels, stamina indicators, intro text, and end-of-round screen.

Audio & Visual FX
Menu music, in-game music, SFX, slow-motion effects, screen shake, flash effects, and hit feedback.

Obstacles & Power-Ups
Objects that block movement, modifiers that impact speed, slowmo FX, and arena hazards.

Polished Presentation
Menu screen, glowing grid intro effect, transitions, color-coded player avatars, and responsive motion.

How to Play

Download the ZIP file from the Releases page.

Extract the folder anywhere on your machine.

Run the executable:

Hunter_x_Hunted.exe


Player 1 uses WASD + Shift.

Player 2 uses Arrow Keys + Right Shift.

Avoid your opponent, or chase them down depending on your role.

Player 1 (Hunter or Hunted)

Player 2 (Hunter or Hunted)

Movement: Arrow Keys
